Less fog. More signal.

The frame is designed for conversations where you want Claude to infer the right working posture, handle uncertainty better, and stay useful while tasks shift between ideation, implementation, review, and refinement.

Intent routing

Encourages the assistant to identify whether the user needs exploration, execution, critique, repair, or synthesis.

Fewer dead stops

Biases toward sensible assumptions and partial progress instead of asking for unnecessary confirmation.

Artifact aware

Supports practical output: files, edits, specs, patches, summaries, and structured deliverables.

Install the skill.

Download the packaged skill from the repository source branch, then upload it to Claude where custom skills are supported.

Download

Get shaping-frame.skill from the main branch.

Upload

Add the skill package to Claude's skill interface, then enable it for the conversations where you want the frame active.

Pair it

For stronger behavior, pair the skill with the repository's custom instruction file.

Tip: keep the repository README nearby. It is the canonical explanation of the frame, its purpose, and expected usage boundaries.

Activate the frame.

Use the CUSTOM_INSTRUCTION.txt file as the companion instruction layer. The skill package supplies the protocol body; the custom instruction helps Claude apply it consistently.

1. Open the repository.
2. Read the README.
3. Download shaping-frame.skill from main.
4. Copy or adapt CUSTOM_INSTRUCTION.txt.
5. Start a conversation and let the frame route the work.
